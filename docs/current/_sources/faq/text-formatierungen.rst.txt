Text-Formatierungen
==============================

.. toctree::
   :maxdepth: 2

Die Texte im Forum, Pinnwand-Posts und in Gebetsanliegen, aber auch in Mails können durch unterschiedliche Formatierungen hervorgehoben und gegliedert werden. Außerdem kannst du Bilder oder Links einfügen.

Allgemeine Formatierungen
-------------------------
::

    [b]Dieser Text ist fett.[/b]
    [i]Dieser Text ist kursiv.[/i]
    [u]So wird eine Überschrift formatiert.[/u]

So sieht's aus:

.. image:: ../_images/faq/bbcode_results_1.png

Aufzählung
----------
::

    Ich liebe Youthweb, weil

    [list]
    [*]ich dort viele Leute treffen kann,
    [*]die meisten von diesen Leuten in wichtigen Dingen ähnlich denken und
    [*]es Spaß macht im Forum seinen Senf dazu zugeben.
    [/list]

So sieht's aus:

.. image:: ../_images/faq/bbcode_results_2.png

Links
-----

Links werden automatisch erkannt und verlinkt. Es ist aber auch möglich, einen bestimmten Text zu verlinken::

    [url=http://youthweb.net]Youthweb[/url]
    [url]http://youthweb.net[/url]

So sieht's aus:

.. image:: ../_images/faq/bbcode_results_3.png

Videos
------

Es können Videos aus Youtube oder Vimeo eingebettet werden. Dazu muss der folgende Code benutzt werden::

    [v]https://youtu.be/mtUVJiDsFvQ[/v]

So sieht's aus:

.. image:: ../_images/faq/bbcode_results_4.png

Bilder
------

Wichtig: Bitte beachte unsere Bilderrichtlinien.

Um Bilder, die irgendwo, z.B. auf eurer eigenen Homepage liegen, anzeigen zu lassen, musst du ihren Speicherort kennen. Da reicht es nicht nur die Homepage anzugeben. z.B. www.youthweb.net, sondern du muss die genau Adresse kennen. z.B. https://support.youthweb.net/fileadmin/user_upload/IMG_1796.jpg und so kann man das Bild dann anzeigen lassen::

    [img]https://support.youthweb.net/fileadmin/user_upload/IMG_1796.jpg[/img]
